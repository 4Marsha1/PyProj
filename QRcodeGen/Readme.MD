# QR Code Generator

## [QRCODE Package](https://pypi.org/project/qrcode/)

### Steps to follow

1. Clone the repository
2. Install the `qrcode` package
   ```
   pip install qrcode
   ```
3. Run the python script
   ```
   python main.py
   ```
4. The generated `QRcode` image will be saved in the directory
