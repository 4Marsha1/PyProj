# Image Enhancer

## [PILLOW PACKAGE](https://pypi.org/project/Pillow/)

### Steps to follow

1. Clone the repository
2. Paste your images in the `./images` directory
3. Install the `Pillow` package
   ```
   pip install Pillow
   ```
4. Run the python script
   ```
   python main.py
   ```
5. The edited images will be stored in the `./editedImages` directory
